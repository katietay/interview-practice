# C++ Problems

Practice problems in C++.

## Problems

1. **Sort with Custom Comparator** - Sorting, structs, comparators
2. **Reverse Words in String** - String manipulation, space handling

## Compiling and Running
```bash
# Compile
g++ -std=c++17 solution.cpp -o solution

# Run
./solution

# Compile and run tests
g++ -std=c++17 test.cpp -o test
./test
```

## Tips

- Use descriptive variable names
- Leverage STL (Standard Template Library)
- Handle edge cases with empty inputs
- Consider memory management
- Use const references for parameters when appropriate
