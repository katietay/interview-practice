Practice problems in Ruby.

## Problems

1. **Character Frequency Counter** - Hash operations, string processing

## Running Tests
```bash
# Run solution file
ruby solution.rb

# Run tests (if using RSpec)
rspec test.rb

# Or run basic test file
ruby test.rb
```

## Tips

- Use descriptive variable names
- Leverage Ruby's enumerable methods (map, select, etc.)
- Hash.new(0) for default values
- Use symbols for hash keys when appropriate
- Handle nil and empty inputs gracefully
